## Setup

### Backend

```
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
flask run
```


### Frontend

```
cd frontend
npm install
npm run dev
```

Create a `.env` file inside the frontend folder and set the backend API URL:

```
VITE_API_URL=http://127.0.0.1:8000
```
